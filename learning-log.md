Today I practiced Python conditionals.

Key rules reviewed:
- if starts a condition chain
- elif checks another condition only if the previous condition was false
- else runs only if all previous conditions were false
- == checks equality
- = assigns a value
- and requires both sides to be true
- or requires at least one side to be true
- not reverses True and False
